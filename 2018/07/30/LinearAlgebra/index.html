Vector and Matrix Algebra

Span

The span of a list of vectors {v1,…, vk} in Rn
is the collection of all linear combinations of {v1,…, vk}. Geometrically, Span{v1,…, vk} consists of all vectors that can be “reached” by taking linear combinations of {v1,…, vk}

Upper and Lower Triangular Matrices

Upper Triangular means that the entries of A “below the diagonal are zero.
Lower Triangular means that the entries of A “above” the diagonal are zero.
not just for SQUARE matrices

Systems of Linear Equations and Nonsigular Matrices

Reduced Row Echelon Form (rref)

Definition

A matrix A is in reduced row echelon form (rref) if each of the following is satisfied:

  1. Any zero-rows occur at the bottom.
  2. The first nonzero entry of a nonzero row is a 1 (called a pivot).
  3. Every pivot occurs to the right of the pivots above it.
  4. All nonpivot entries in a column containing a pivot are zero.

Rank and Nullity

Rank : the number of pivot columns. –> dependent variables
Nullity: the number of nonpivot columns. –> independent variables (free)

rref Systems

Consistent and Inconsistent

consistent (or solvable): the augmented column contains no pivot.
inconsistent (or unsolvable): the augmented column contains a pivot.

Number of Solutions

A consistent rref system has:
exactly one solution if it has no free variables. This means that
rank = number of variables and nullity = 0
infinitely many solutions if it has free variables. This means that
rank < number of variables and nullity > 0
An inconsistent system has no solutions.

Gauss-Jordan Elimination

  1. If aij = 0, switch the ith row with the first row below it whose jth entry is nonzero. If no such row exists, increase j by one and repeat this step.
  2. Multiply the ith row by 1/aij to make a pivot entry in the position (i,j).
  3. Eliminate all other entries in the jth column by subtracting suitable multiples of the ith row from the other rows.
  4. Increase i and j by one and return to Step 1.
    The algorithm terminates after the last row or last column is processed. The output of this algorithm is the reduced row echelon form rref(A) of A.

Nonsingular Matrices

x=A-1b

inverse / reciprocal. 逆
可逆性 A n × n matrix A is invertible / nonsingular if and only if rank(A) = n .

Gauss-Jordan Algorithm for Computing A-1
Let A be a n × n matrix. The following steps will compute A-1 or determine that A-1exists.

  1. Form the augmented matrix [A | IAn].
  2. Use Gauss-Jordan elimination to row reduce [A | In] to [rref(A) | B].
  3. If rref(A) ̸= In, then A is not invertible.
  4. If rref(A) = In, then A is invertible and B = A-1.

Matrix Inverse Rules
Involution Rule: If A is invertible, then (A-1)-1 = A.
Transposition Rule: If A is invertible, then (A-1) = (A )-1.
Reverse Multiplication Rule: If A and B are invertible, then (AB)-1 = B-1A-1.
Scaling Rule: If A is invertible and c ̸= 0, then (c · A)-1 = (1/c) · A-1.

Linear Independence

Let {v1, v2,…, vk} be the columns of a matrix A = [v1, v2,…, vk]
The list of vectors {v1, v2,…, vk} is
linearly independent if rank(A) = the number of columns of A
linearly dependent if rank(A) < the number of columns of A
A matrix A whose columns are linearly independent (rank(A) equals the number of columns of A) is said to have full column rank.

Nonsingularity of ATA
If A has full column rank, then ATA is nonsingular.
This means that the system ATAx = ATb has a unique solution x , even if Ax = b has no solution.

The Method of Least Squares

When Ax = b has no solution,multiplyby AT and solve ATAx =ATb for x .
The vector x is called the least squares approximate solution to Ax = b .
This approximate solution makes the error E = ∥Ax − b∥A2 as small as possible.

Gram-Schmidt Orthogonalization and QR-Factorization

Inversion: (AB)T = BTAT
Transposition: (AB)-1 = B-1A-1
orthogonal.
orthonormal.

Orthonormal List of Vectors
A list of vectors {q1, q2,…, qk} is called orthonormal if each vector is a unit vector and each pair of vectors is orthogonal.

Matrices with Orthonormal Columns
Matrices with orthonormal columns satisfies QTQ = I . (proved by its definition)
Note that these matrices need not be square.
But when Q is a square matrix, the relation QTQ = I means that QT = Q-1 . Such matrices are called orthogonal matrices.
如果正方形矩阵的列向量是orthonormal（相互orthogonal且长度都为1）的，则该矩阵是orthogonal的。

Least-Squares Approximationof Qx = b
The least-squares approximation x of Q x = b is x = QTb .
Consequently, if Qx = QQTb != b then the system Qx = b is inconsistent.

QR-Decompositions
Given A = QR , the least squares approximate solution to Ax = b is the solution x to the upper-triangular system Rx = QTb .

Using Gram-Schmidt Algorithm to Compute A=QR
Input: A matrix A with linearly independent columns (rank(A) equals the number of
columns of A).
Output: A QR-decomposition A = QR .

Eigenvalues & Eigenvectors

Eigenvalues. 特征值
递推关系
Eigenvalues
A scalar λ is an eigenvalue of an n×n matrix A if the matrix A−λ·In is singular.

  • Calculate λ: |A − λ · In| = 0
  • Test to Determine if λ is an Eigenvalue of A: a scalar λ is an eigenvalue of an n × n matrix A if and only if rank(A − λ · In) < n .

Eigenvectors
Suppose that λ is an eigenvalue of a matrix A. An eigenvector of A associated to the eigenvalue λ is a vector v satisfying the equation Av = λ·v

  • Test to Determine if v is an Eigenvector of A: A vector v != 0 is an eigen vector of A if and only if Av =λ·v for some scalar λ. In this situation, λ is an eigenvalue of A.

    Eigenpair
    An eigenpair of a matrix A is a pair (λ,v) where λ is an eigen value of A and v is an eigenvector of A corresponding to the eigenvalue λ.
    many different eigenvectors may be associated to the same eigenvalue.

Pivot Basis

Geometric Multiplicity
The geometric multiplicity of λ as an eigenvalue of A is a number that measures the number of linearly independent(nonpivot columns) eigenvectors of A associated to λ. gmA(λ) = nullity(A−λ·In) .

The Pivot Basis Associated to an Eigenvalue
Suppose that λ is an eigenvalue of an n×n matrix A. To find the pivot basis associated to λ, start by considering the system of linear equations given by (A − λ · In) x = 0 .
Use the reduced row echelon form rref[A − λ · In | 0 ] to write the general solution to (A−λ·In)x = 0 as x = c1·v1 + c2·v2 +···+cg·vg . The list of vectors {v1, v2,…, vg} is called the pivot basis associated to λ. (but not the eigenvector associated to λ)

Eigenvectors are Linear Combinations of the Pivot Basis
The eigenvectors of A associated to λ are the linear combinations of the pivot basis {v1, v2,…, vg}. That is, every eigenvector of A associated to λ is of the form v = c1 · v1 + c2 · v2 + · · · + cg · vg where c1,c2,…,cg are scalars.

Diagonalization

Diagonalizable
An n × n matrix A is diagonalizable if it is possible to find an invertible n × n matrix P and an n × n diagonal matrix D satisfying A = PDP−1 .
The matrix power Ak is given by Ak = PDkP−1

  • A is diagonalizable if and only if the sum of the geometric multiplicities of each eigenvalue of A is equal to n.

Algorithm for Diagonalizing a Matrix
Let A be a diagonalizable matrix. To find a diagonalization A = PDP−1 ., start with two empty n × n matrices P and D. Then complete the following steps for each eigenvalue λ of A.

  1. Find the pivot basis {v1, v2,…, vg} associated to λ.
  2. Insert each vector from the pivot basis { v 1, v 2, . . . , v g} into the first available columns of P.
  3. Insert λ onto the first available diagonal entries of D a total number of gmA(λ) times.

Spectral Theorem and Principal Component Analysis

symmetric: AT = A
orthogonal: QT = Q<-1

Spectral Theorem (Diagonalization of Symmetric Matrices)
Let A be a symmetric matrix, so AT = A . Then there is a diagonalization A = QDQT where Q is an orthogonal matrix QT= Q-1 .
The spectral theorem guarantees that any given symmetric matrix is diagonalizable and that the eigenvectors chosen during the diagonalization process may be chosen to be or- thonormal.
作用：如果能够找到orthogonal的Q，就无需求Q-1了，直接求QT就好。

The Covariance Matrix
A -> Ac: centered data.
when we use a line to approximate centered data, the line is always of the form f (x) = m · x. That is, the line always passes through the origin.

Suppose that data is stored in an n × p matrix A, where each column corresponds to a variable. The covariance matrix of A is defined as the new matrix Σ (sigma) given by
Σ= 1/(n-1) AcT Ac
Features : positive definite

  • symmetric: ΣT = Σ -> spectral theory -> orthonormal eigenvectors
  • eigenvalues of the covariance matrix are positive numbers

Orthogonal Distance

  • least squares: use vertical distance to measure the error (linear regression)
  • principal component analysis (PCA): use orthogonal distance to measure the error (orthogonal regression)

The Orthogonal Error in Approximating Centered Data by f (x) = m · x
Consider the problem of using a line f(x) = m · x to approximate a centered data set, whose points are listed as vectors { p1, p2, . . . , pn}. The orthogonal error in using f (x) = m · x to approximate the data set is the sum of the squares of the orthogonal
distances from each of these points to the vector w = ⟨1, m⟩.
That is, the orthogonal error is
E=∥p1 −projw(p1)∥2 +∥p2 −projw(p2)∥2 +···+∥pn −w(pn)∥2

Orthogonal Regression

  • Goal: Given a data set, find the line that minimizes the orthogonal error.

  • First Principal Component: Suppose that λ1 is the largest eigenvalue of the covariance matrix Σ(which is positive definite). A first principal component of Σ is an eigenpair (λ1, v1) of Σ.

  • Technique: Let A be a matrix whose rows consist of data points. Let (λ1, v1) be a first principal component of the covariance matrix Σ. Write the vector v1 in coordinates v1 = ⟨x, y⟩ and define m = y/x . Then the line f (x) = m · x minimizes the orthogonal error.
